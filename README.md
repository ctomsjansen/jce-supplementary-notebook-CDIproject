# jce-supplementary-notebook-CDIproject
Supplementary Jupyter Notebook for CDI education project. The article "Capacitive deionization for water desalination: A hands-on experimental engineering project" is published in the Journal of Chemical Education. 

This repository contains the Jupyter Notebook supplementary material for:
- [Article] Capacitive deionization for water desalination: A hands-on experimental engineering project  
- [Autors] C.J.E. Tomsjansen, S.U. Khan, N.J.H. Boon, H.B. Eral
- [Journal name] Journal of Chemical Education, 2026

## Contents
- `Jupyter Notebook file - Water_for_all.ipynb`: main supplementary notebook
- `adsorptie-resultaten.csv`: example (real) adsorption data. The data could be used to fit model equations through it.
- `desorptie-resultaten.csv`: example (real) desorption data. The data could be used to fit model equations through it.

## How to use
Download the repository or open the notebook in Jupyter Notebook/JupyterLab.
